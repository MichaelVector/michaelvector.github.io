黑苹果开启原生NTFS读写

发布日期：2018年8月15日

最后更新：2020年2月28日

黑苹果

其实苹果系统是支持原生NTFS读写的，因为版权的问题不能开放给普通用户授权使用。但默认是支持的，我们可以通过“旁门左道”来实现。

一、打开Bash（终端）——查看所有分区的卷标

输入diskutil list查看所有硬盘的分区列表

二、编辑fstab文件（/etc/fstab）

输入sudo nano /etc/fstab在输入密码回车进行配置

根据自己要配置的NTFS分区卷标输入配置信息

  • LABEL=D none ntfs rw,auto,nobrowse
  • LABEL=E none ntfs rw,auto,nobrowse
  • LABEL=F none ntfs rw,auto,nobrowse
  • LABEL=VectorW none ntfs rw,auto,nobrowse

输入盘符以后按Ctrl+X进行保存，在输入Y进行确认，之后回车确认写入文件/etc/fstab保存文件

因为加入了nobrowse所以Finder中看不到修改过的NTFS分区（不加入nobrowse不会生效）所以要使用快捷方式进行访问。
用磁盘工具将配置好的NTFS分区卸载再装在使配置生效（无需重启）

三、生成快捷方式

在终端中输入
sudo ln -s /Volumes/D ~/Desktop/D
即可在桌面生成NTFS分区D的快捷方式
